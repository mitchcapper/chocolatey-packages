## Chocolatey Packages
These are [Chocolatey](https://chocolatey.org/) packages to install the projects listed below that I maintain.

## Installation
1. Install [Chocolatey](https://chocolatey.org/) *(if not already installed)*.
2. `choco install <package name>`.
